gulpメモ

・「https://nodejs.org/ja」より「Node.js（推奨版）」をダウンロード
・「Node.js」をインストール（すでにインストールしている場合は不要）

・VSCodeのターミナルで作業フォルダを開く
・VSCodeのターミナルで「npm init -y」と入力し「package.json」を作成

・VSCodeのターミナルで「npm install -D gulp」と入力しgulpプラグインをインストール
・VSCodeのターミナルで「npm install -D sass」と入力しDartSass対応化プラグインをインストール
・VSCodeのターミナルで「npm install -D gulp-sass」と入力しSassをコンパイルするプラグインをインストール
・VSCodeのターミナルで「npm install -D gulp-autoprefixer」と入力しベンダープレフィックスを自動付与するプラグインをインストール
・VSCodeのターミナルで「npm install -D gulp-cssnano」と入力しSassをミニファイ化するプラグインをインストール
・VSCodeのターミナルで「npm install -D gulp-rename」と入力しリネームプラグインをインストール
・VSCodeのターミナルで「npm install -D gulp-sourcemaps」と入力しソースマップを作成するプラグインをインストール
・VSCodeのターミナルで「npm install -D gulp-uglify」と入力しJSをミニファイ化するプラグインをインストール
（「npm install -D gulp sass gulp-sass gulp-autoprefixer gulp-cssnano gulp-rename gulp-sourcemaps gulp-uglify」で一括インストール）

・新しいテキストファイルで「gulpfile.js」を作成、内容を他の「gulpfile.js」からコピーする
・新しいテキストファイルで「.browserslistrc」を作成、内容を他の「.browserslistrc」からコピーする

・VSCodeのターミナルで「npx gulp」と入力しgulpを起動する
