progate PHPⅠ

PHPとは？

echo

PHPはHTMLに埋め込んで使うことができる
PHPでは文末にセミコロン「;」を使って文を区切る
文字列の出力は「echo」
「7 * 2」のように計算もできる
演算子「+　-　*　/　%(割り算した余り)」など

<?php echo'<h2>初級</h2>'; ?>
<?php
 echo'こんにちは';
 echo'PHPを学ぼう';
?>

PHPの基礎を学ぼう

データの種類

データ型は文字列と数値の2種類がある

<?php
 echo 5 + 2; //結果:7 (数値)
 echo '5 + 2'; //結果:5 + 2 (文字列)
?>

変数

変数は「$変数名 = 値;」で定義できる
変数名の付け方／〇：$date $userName、×：$1name(数字開始) $namae(ローマ字) $名前(日本語)

<?php
 $name = 'Bob';
 echo $name; //結果:Bob
?>

変数の値を更新する

変数は後に代入されたものに上書きされる
「$x = 1; $x = $x + 10;」のように足し続けることもできる
「$x = $x + 10;」は「$x += 10;」のように省略することもできる
数字が1の時だけ「++」でさらに省略できる、「++$x」で処理の前に、「$x++」で処理の後に足すことができる、「--」も同様

<?php
 $x = 1;
 $x = $x + 10; //結果:11
?>
<?php
 $x = 3;
 echo ++$x; //結果:4 (echoの前に+1される)
 echo $x++; //結果:3 (echoの後に+1される)
?>

文字列の連結

「.」で文字列を連結できる（「echo $lesson.$level;」）
「.=」を用いれば変数と文字列の連結を省略できる
ダブルクオーテーション「"..."」で囲った文字列の中に「{...}」で囲んだ変数を入れると変数展開することができる（※シングルクオーテーション「'...'」では変数展開されないので注意）

<?php
 $lesson = PHP;
 echo $lesson.'入門'; //結果:PHP入門
?>
<?php
 $name = 'にんじゃ';
 $name .= 'わんこ';
 echo $name; //結果:にんじゃわんこ
?>
<?php
 $name = 'にんじゃわんこ';
 echo "こんにちは、{$name}さん" //結果:こんにちは、にんじゃわんこさん
?>

条件分岐

if文

<?php
 $x = 20;
 if ($x > 10) {
  echo $xは10より大きい;
 }
?>

else, elseif文

比較演算子「<　<=　>　>=　==　!=(等しくない)」

&&, ||

「&&」は「かつ」、「||」は「または」という論理演算子
「!」は演算の真偽が逆になる

switch文

下記のようにif, elseifをシンプルに記すことができる

<?php
 if ($coin == 0) {
  echo '表';
 } elseif ($coin == 1) {
  echo '裏';
 } else {
  echo 'エラー';
 }
?>
<?php
 switch ($coin) {
  cose 0:
   echo '表';
   break;
  cose 1:
   echo '裏';
   break;
  default:
   echo 'エラー';
   break;
 }
?>

配列を使ってみよう

配列

<?php
 $names = array('John','Kate','Bob');
 $names[] = 'Mary'; //配列の末尾に値を追加
 echo $names[3]; //結果:Mary
 $names[1] = 'Jane'; //値の上書き
 echo $names[1]; //結果:Jane
?>

連想配列

配列と違いキーと値で保存できる

<?php
 $user = array(
  'name' => 'わんこ',
  'age' => '14',
  'gender' => 'mail'
 );
 echo $user[name]; //結果:わんこ
 $user['level'] = 'beginner';//キーと値を追加
?>

繰り返し処理

for文

<?php
 for ($i = 1; $i <= 100; $i ++){
  echo $i;
 }
?>

while文

条件式を指定し、それがtrueの間、処理が繰り返される
（「$i++;」を入れないと無限ループになりPCに極端に負荷をかけてしまうため、どこかで「false」になるよう気を付ける）

<?php
 $i = 1;{
 for ($i <= 100){
  echo $i;
  $i++;
 }
?>

break文

現在のループを強制的に中断する命令

<?php
 for ($i = 1; $i <= 10; $i ++){
  if ($i > 5) {
   break;
  }
 }
 echo $i; //結果:12345
?>

continue文

現在の周だけをスキップ

<?php
 for ($i = 1; $i <= 10; $i ++){
  if ($i % 3 == 0) {
   continue;
  }
 }
 echo $i; //結果:12457810 (3の倍数を除外)
?>

foreach文

配列、または連想配列に対して先頭データから順に繰り返し処理させる

<?php
 $towns = array('東京','大阪','京都');
 foreach ($towns as $town){
  echo $town.' ';
 }
 echo $town; //結果:東京 大阪 京都
?>
<?php
 $colors = array(
  'Apple' => 'Red',
  'Banana' => 'Yellow',
  'Grape' => 'Purple'
 );
 foreach ($colors as $key => $value){
  echo $key.':'.$value.' ';
 }
 echo $town; //結果:Apple:red Banana:Yellow Grape:Purple
?>

関数を使ってみよう

関数

関数とは、あるまとまった処理を行い、値を返すもの
PHPにもとから組み込まれている関数を、組み込み関数と言う
「()」の中に与える値を「引数」という
関数例：strlen／文字列の文字数を返す、count／配列の要素の数を返す、rand／指定引数内のランダムな整数を返す

<?php
 $language = 'PHP'
 echo strlen($language); //結果:3 (文字列の文字数が返される)
?>

関数を自作する

自分で関数を作成することもできる
関数定義の「()」は仮引数と言う

<?php
 function printCircleArea($radius){
  echo $radius * $radius * 3;
 }
 printCircleArea(3); //結果:27
?>

戻り値

関数は値を「返す」ことができる、この値のことを戻り値と呼ぶ

<?php
 function getSum($num1,$num2){
  return $num1 + num2;
 }
 $sum = getSum(1,3);
 echo $sum; //結果:4
?>
